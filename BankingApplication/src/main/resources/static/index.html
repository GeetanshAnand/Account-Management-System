Project Summary

This Banking Application allows users to manage bank accounts seamlessly. Users can add new accounts, view all existing accounts, search for specific accounts, and perform operations such as deposits, withdrawals, and account deletions. It is built using Java, Spring Boot, and an interactive front-end for a smooth user experience.
